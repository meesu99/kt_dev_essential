🚀 트랜잭션 연습 프로젝트

📚 프로젝트 소개

이 프로젝트는 Spring Framework의 트랜잭션 관리 기능을 학습하기 위한 예제입니다.

  • 트랜잭션이 없는 경우의 위험성
  • 트랜잭션이 있는 경우의 안전성
  • H2 데이터베이스를 활용한 실습

🔗 바로가기

학생 관리 시스템 트랜잭션 테스트

📖 사용법

  1. 학생 관리 시스템에서 학생 목록을 확인합니다.
  2. 트랜잭션 테스트 페이지에서 다양한 시나리오를 테스트합니다.
  3. H2 콘솔(http://localhost:8082)에서 데이터베이스 상태를 확인합니다.
